# components/panels

属性面板子组件。

## 文件说明

- **PanelNoSelection.vue** — 无选中对象时的侧栏面板（线路管理、排版控制）
- **PanelNetworkStats.vue** — 线网统计仪表盘（线路总数、车站总数、换乘站数量、总里程、平均站间距、最长/最短线路、最大换乘站、各线路排行）
- **PanelStationSingle.vue** — 单个站点属性编辑面板
- **PanelStationMulti.vue** — 多个站点批量编辑面板
- **PanelEdgeSingle.vue** — 单条线段属性编辑面板
- **PanelEdgeMulti.vue** — 多条线段批量编辑面板
- **PanelAnchor.vue** — 锚点属性编辑面板
