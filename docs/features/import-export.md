# 导入导出

Metro Studio 支持多种数据格式的导入和导出。

![导出界面](/images/export.png)

## 导入

### 从 OpenStreetMap 导入

直接从 OSM 导入真实的地铁线网数据，自动解析线路和站点信息。

### 导入项目文件

导入之前保存的 ZIP 格式项目文件，恢复完整的编辑状态。

## 导出

### PNG 图片

导出高清 PNG 格式的线路图，适合分享和打印。

### 时间线视频

将线网发展历程导出为动画视频，适合演示和教学。

### 项目文件

导出 ZIP 格式的项目数据，包含所有站点、线路和配置信息，方便备份和迁移。
