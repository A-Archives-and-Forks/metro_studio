# 地图编辑

在真实的 OpenStreetMap 地图上绘制地铁线网。

![编辑器界面](/images/editor.png)

## 基础编辑

- **添加站点**：点击地图即可在精确地理位置添加站点
- **连接线路**：拖拽站点之间的连线，创建线路
- **编辑属性**：选中对象后在右侧面板编辑名称、颜色、样式等

## 高级编辑

- **框选**：拖拽选择区域内的多个对象
- **多选**：Shift + 点击添加到选择
- **整条线路选择**：Alt + 点击线段选中整条线路
- **批量编辑**：选中多个对象后统一修改属性

## 线路管理

- 创建多条线路，设置不同颜色和样式
- 标记线路开通年份
- 区分运营中 / 在建 / 规划状态
