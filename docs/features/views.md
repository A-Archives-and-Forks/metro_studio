# 多视图展示

Metro Studio 提供四种视图，满足不同场景需求。

## 地图视图

在 OpenStreetMap 底图上进行真实地理位置编辑，精确定位每个站点。

![编辑器界面](/images/editor.png)

## 示意图视图

官方风格的八方向规整线路图，适合展示和导出。

![示意图视图](/images/schematic.png)

## 车载 HUD 视图

模拟地铁车厢内的显示屏效果，动态显示当前站和下一站信息。

![HUD 视图](/images/hud.png)

## 时间线视图

按年份筛选线路，播放线网发展动画，直观展示城市轨道交通的发展历程。

![时间线视图](/images/timeline.png)
