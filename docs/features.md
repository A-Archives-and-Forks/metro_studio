# 功能总览

Metro Studio 提供从地图编辑到导出分享的完整工作流。

![编辑器界面](/images/editor.png)

## 核心功能

- **地图编辑** — 在 OpenStreetMap 上绘制地铁线网，支持框选、多选、批量编辑
- **示意图生成** — 一键生成八方向规整的官方风格示意图
- **多视图展示** — 地图视图 / 示意图视图 / 车载 HUD 视图 / 时间线视图
- **AI 辅助命名** — 根据地理信息自动生成中英文站点名称
- **时间线动画** — 按年份展示线网发展历程，支持视频导出
- **数据导入导出** — OSM 导入、PNG/视频导出、项目文件导入导出

详细说明请查看各功能子页面。
