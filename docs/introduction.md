# 项目简介

**Metro Studio** 是一个专为轨道交通爱好者和城市规划者设计的 Web 应用，让用户能够在真实地图上轻松绘制专业的地铁线路图。

![编辑器界面](/images/editor.png)

## 核心定位

- **目标用户**：轨道交通爱好者、城市规划者、交通设计师
- **应用类型**：基于 Web 的单页应用（SPA）
- **在线访问**：[metro-studio-iota.vercel.app](https://metro-studio-iota.vercel.app/)
- **开源协议**：GPL-3.0

## 为什么选择 Metro Studio

| 对比项 | Metro Studio | 传统工具 |
|--------|-------------|---------|
| 真实地图编辑 | ✅ OpenStreetMap | ❌ 纯画布 |
| 自动示意图 | ✅ 一键生成 | ❌ 手动绘制 |
| 多视图 | ✅ 4 种视图 | ❌ 单一视图 |
| AI 辅助 | ✅ 智能命名 | ❌ 无 |
| 数据导入 | ✅ OSM 导入 | ❌ 手动输入 |
| 价格 | ✅ 免费开源 | 💰 部分付费 |

## 技术栈

Vue 3 · Pinia · MapLibre GL JS · Turf.js · Vite · OpenStreetMap
